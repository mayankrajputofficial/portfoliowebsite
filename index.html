About

I'm a Cloud & DevOps Engineer building and running production systems on Oracle Cloud Infrastructure. After leaving a corporate job, I made a deliberate pivot into infrastructure and automation — learning by doing, not just watching tutorials.

My current setup: a self-hosted portfolio running on an OCI Always Free VM (1 CPU, 1 GB RAM) — containerised with Docker, served by Nginx, deployed automatically via GitHub Actions CI/CD. Every piece of the stack is something I debugged, fixed, and understood — from iptables rules to SSH key permissions.

I document everything I learn on my blog — real debugging stories, real infrastructure lessons. No fluff. I'm actively working toward OCI certifications and deeper Kubernetes & Terraform experience.
